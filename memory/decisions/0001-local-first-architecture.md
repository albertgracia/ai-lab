# Decision 0001 - Local First Architecture

Date: 2026-05-09

Decision:
AI-Lab will prioritize local execution over cloud APIs.

Reason:
- Privacy
- Control
- Lower dependency on external providers
- Better homelab learning value
- Future GPU cluster expansion

Current inference backends:
- LM Studio
- Ollama

Future:
- Dedicated gaming PC inference nodes
- Multi-model routing
- Distributed cognitive runtime
