# AI-Lab Architecture Memory

AI-Lab is a local cognitive AI infrastructure.

Current components:
- Ubuntu Server VM on Hyper-V
- Docker
- Traefik
- Portainer
- Open WebUI
- Ollama
- LM Studio integration
- Qdrant vector database
- .agent cognitive framework
- Semantic search runtime
- Prompt builder
- LLM invocation runtime
- Agent router
- OpenCode local coding interface

Design principles:
- Local first
- Modular runtime
- Safe tool execution
- Git-tracked infrastructure
- Semantic retrieval
- Agent orchestration
- Persistent memory
